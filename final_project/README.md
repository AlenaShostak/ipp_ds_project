# Прогноз выявляемости ИППП

## Описание проекта
Проект посвящён анализу и прогнозированию выявляемости
инфекций, передающихся половым путём (ИППП), с использованием
методов Data Science и машинного обучения.

## Данные
Используются агрегированные обезличенные медицинские данные
за период 2022–2025 гг.
Данные представлены в разрезе месяцев, пола пациентов
и типа инфекции.

## Цель проекта
Целью проекта является выявление сезонных закономерностей
и построение модели прогнозирования количества выявленных
случаев ИППП.

## Используемые методы
- разведочный анализ данных (EDA)
- анализ сезонности
- визуализация данных
- машинное обучение (регрессия)

## Модели
- Linear Regression
- Random Forest Regressor

## Метрики качества
- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)
